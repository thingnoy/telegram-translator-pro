### Naming Convention:
```

Examples:
```

### How to Add Screenshot:
1. Take screenshot of the script in action
2. Resize/crop to show key features (1200x800 recommended)
3. Save as PNG with proper naming convention
4. Add to this README with description
5. Update main README.md and scripts/README.md with new screenshot
